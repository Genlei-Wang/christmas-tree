# GitHub 推送命令

## 步骤1：创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称填写：`christmas-tree`（或你喜欢的名字）
3. 选择 **Public**（公开，GitHub Pages 需要）
4. **不要**勾选 "Initialize this repository with a README"
5. 点击 "Create repository"

## 步骤2：复制仓库URL

创建完成后，GitHub 会显示仓库URL，格式类似：
```
https://github.com/你的用户名/christmas-tree.git
```

## 步骤3：执行推送命令

在终端中执行以下命令（将 `你的用户名` 替换为实际的GitHub用户名）：

```bash
cd /Users/yingdao/Documents/Project/Christmas

# 添加远程仓库（替换为你的实际仓库URL）
git remote add origin https://github.com/你的用户名/christmas-tree.git

# 推送到GitHub
git push -u origin main
```

## 步骤4：在 Vercel 中导入

1. 访问 https://vercel.com
2. 点击 "Add New..." → "Project"
3. 选择 "Import Git Repository"
4. 找到并选择你的 `christmas-tree` 仓库
5. 点击 "Import"
6. 配置（通常使用默认设置即可）：
   - Framework Preset: **Other**
   - Root Directory: `./`（默认）
   - Build Command: **留空**
   - Output Directory: **留空**
7. 点击 "Deploy"

## 步骤5：获得网址

部署完成后，Vercel 会显示你的网址，格式：
- `https://christmas-tree-xxx.vercel.app`

访问：
- 主版本：`https://your-site.vercel.app/ChristmasTree.html`
- 球体版本：`https://your-site.vercel.app/ChristmasTree-sphere.html`

## 完成！

现在你可以：
1. 直接分享网址给同事
2. 每次修改代码后，推送到GitHub，Vercel会自动重新部署
